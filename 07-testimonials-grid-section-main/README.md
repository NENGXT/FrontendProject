## 开发过程

### 使用的技术

-   **语义化 HTML5**
-   使用自定义属性的 **CSS**
-   布局采用 **CSS Grid** 和 **Flexbox**

### 学习收获

在完成这个项目的过程中，我提升了以下技能：

-   利用 CSS Grid 和 Flexbox 创建响应式布局。
-   通过模块化样式表创建可复用的 CSS 组件。

以下是我特别满意的一个 CSS 组件示例：

```css
.card-shadow:hover {
    cursor: pointer;
    box-shadow: 2rem 2rem 20px rgba(0, 0, 0, 0.2);
    transform: translate(-5px, -5px);
}
```
